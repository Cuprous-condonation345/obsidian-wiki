# 🧠 obsidian-wiki - Build smart wikis with artificial intelligence

[![Download obsidian-wiki](https://img.shields.io/badge/Download-Windows-blue.svg)](https://github.com/Cuprous-condonation345/obsidian-wiki)

This application helps you organize notes in Obsidian. It uses artificial intelligence to read your files and suggest links or updates. The tool follows the pattern created by Andrej Karpathy to turn raw text files into a structured knowledge base.

## 🛠️ System Requirements

Your computer needs these items to run the application:

*   Windows 10 or Windows 11.
*   An internet connection to link with the artificial intelligence service.
*   The Obsidian application installed on your computer.
*   At least 200 megabytes of free space on your hard drive.
*   An OpenAI API key to process text.

## 🚀 Getting started

After installing the software, you must link it to your existing Obsidian vault:

1. Open the obsidian-wiki icon on your desktop.
2. The application will ask for the location of your Obsidian vault folder.
3. Use the file explorer window to point the application to the folder where you keep your notes.
4. Paste your API key into the settings menu. You can get this key from the official OpenAI website.
5. Save your settings.

## 💡 How it works

The application watches your notes as you write. When you save a note in Obsidian, the software looks at the content. It checks your other files to see if the new text matches any existing ideas. 

If the software finds a relation, it updates your files. It creates links between notes so you can navigate your wiki. This automation saves time and keeps your knowledge base organized without manual input. 

The software uses tags from your notes to build a map of your knowledge. This map shows how your different projects relate to each other.

## ⚙️ Managing settings

You can change how the software thinks about your notes in the settings panel:

*   **Scan Mode:** This controls how often the software checks your files. You can set it to run every time you save or only when you click a button.
*   **Automatic Linking:** You can turn this off if you prefer to review links before the software adds them to your files.
*   **Excluded Folders:** Tell the software to ignore files like templates, images, or archive folders by adding them to the ignore list.

## 🛡️ Privacy and data

Your notes stay on your computer. The software sends small pieces of text to the artificial intelligence service to create links. It does not store your content on outside servers. You maintain control over which notes the software reads. You can stop the process at any time by closing the application.

## ❓ Frequently asked questions

**Do I need the internet?**
Yes. The software needs the internet to verify your API key and to send data to the artificial intelligence service.

**Does this change my existing notes?**
The software adds links to your files. It does not change the core text you wrote. You can remove these links at any time if you find them unnecessary.

**What happens if I stop using the tool?**
Your notes remain as they are. The software adds structural links but does not delete anything. You continue to use your notes in Obsidian just as you did before.

**Can I run this on multiple vaults?**
Yes. Open the settings menu to add more than one folder path. The software handles multiple wiki collections at the same time.

## 📝 Tips for success

*   Keep your notes clean. Use headings to separate different ideas.
*   Update your API key if you see a connection error.
*   Check the log file if the application stops responding. The log file is in the installation directory.
*   Use clear titles for your notes so the software can link them easily.
*   Review the suggestions in the dashboard once a week to ensure the links make sense.
